# artsey.io based firmware for ergohaven/macropad

Compatible with Vial.

## Ergohaven M4CR0Pad

A RP2040-powered Macropad with a 3x4 layout.

### Macros

*M0* - `Cmd + C` \
*M1* - `Cmd + V` \
*M2* - `Cmd + X` \
*M3* - `Cmd + Z` \
*M4* - `Cmd + Shift + 4` \
*M5* - `Cmd + T` \
*M6* - `Cmd + W` \
*M7* - `Cmd + F` \
*M8* - `Cmd + S` \
*M9* - `Cmd + B` \
*M10* - `Cmd + R` \
*M11* - `Cmd + O` \
*M12* - `Cmd + A` \
